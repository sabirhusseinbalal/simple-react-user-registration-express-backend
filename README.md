# React User Registration Form with Express Backend

A responsive user registration form built with React and styled using Tailwind CSS. The form validates user input with react-hook-form and communicates with an Express backend to handle submissions.


## Technologies Used
- **React**
- **Tailwind CSS**
- **Express.js**
- **Axios**
- **react-hook-form**

## Features
- User input validation
- Responsive design
- Integration with a backend API
- Error handling for existing usernames


### Prerequisites
- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Installation

1. Clone the repository.

    ```bash
   git clone https://github.com/sabirhusseinbalal/simple-react-user-registration-express-backend.git
   ```
  
2. Run `npm install` to install dependencies.

### 3: Run (Frontend Terminal)
```bash
npm run dev
```

### 4: Run (Backend Terminal)
```bash
npm start
```
